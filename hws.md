# Xcode运行c project
* 1.打开Xcode，创建c语言方案；
* 2.点击左上角File-New-Project
* 3.选择创建的方案类型：选择OS X——Application——Command Line Tool 点击Next。
* 4.完善创建的方案信息，最后一项要选择c语言。
* 5.选择文件的保存位置；
* 6.创建完成后，点击左侧main.c进入编辑区域。
